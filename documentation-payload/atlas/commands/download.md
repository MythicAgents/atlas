## Summary

Download a file from the victim machine to the Mythic server in chunks (no need for quotes in the path).

- Needs Admin: False  
- Version: 1  
- Author: @Airzero24 

### Arguments

#### file_path

- Description: Path to remote file to be downloaded  
- Required Value: True  
- Default Value: None  

## Usage

```
download [path to remote file]
```
