## Summary

Kills a running job and removes it from the atlas instance's list of running jobs. You can retrieve a list of running jobs with the `jobs` command.
- Needs Admin: False  
- Version: 1  
- Author: @Airzero24

### Arguments

#### job_id

- Description: The Job Id for the running job to be killed  
- Required Value: True  
- Default Value: None  

## Usage

```
jobkill [job id]
```
