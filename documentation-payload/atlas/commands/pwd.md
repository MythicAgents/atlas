## Summary

Get current working directory of the Atlas instance. 
- Needs Admin: False  
- Version: 1  
- Author: @Airzero24

## Usage

```
pwd
```

## Detailed Summary
This command uses the `System.IO.Directory.GetCurrentDirectory` class method to retrieve the current running process's working directory.
